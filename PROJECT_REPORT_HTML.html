1. סקירה כללית

נבנה אתר תדמיתי מלא עבור ארגון "ועד מבקשי ה'", הכולל:

  • ✅ אתר ציבורי עם כל העמודים הנדרשים
  • ✅ מערכת ניהול תוכן מלאה (Admin Panel)
  • ✅ מסד נתונים PostgreSQL
  • ✅ העלאת תמונות וסרטונים
  • ✅ טופס יצירת קשר
  • ✅ אינטגרציה עם נדרים פלוס לתרומות

2. מה נבנה

Frontend (חלק המשתמש)

עמודים ציבוריים:

  1. עמוד הבית - וידאו רקע, סטטיסטיקות, כפתורי CTA
  2. אודותינו - מידע מפורט על הארגון
  3. גלריה - תמונות וסרטונים עם סינון
  4. צור קשר - טופס יצירת קשר
  5. תרומה - קישור לנדרים פלוס

מערכת ניהול (Admin Panel):

  1. התחברות מנהל - מסך login
  2. לוח בקרה - סטטיסטיקות
  3. ניהול גלריה - העלאת תמונות/סרטונים
  4. ניהול הודעות - צפייה וניהול הודעות

3. טכנולוגיות

Frontend

  • React 18 - ספריית UI מודרנית
  • TypeScript - תכנות בטוח
  • Vite - כלי בנייה מהיר
  • Tailwind CSS - עיצוב מהיר
  • Framer Motion - אנימציות
  • React Router - ניווט
  • React Hook Form + Zod - טופסים עם ולידציה
  • Axios - תקשורת עם שרת

Backend

  • Node.js 20 - סביבת הרצה
  • Express.js - שרת API
  • Prisma ORM - ניהול מסד נתונים
  • PostgreSQL 15 - מסד נתונים
  • JWT - אימות משתמשים
  • bcryptjs - הצפנת סיסמאות
  • Cloudinary - אחסון מדיה
  • Multer - העלאת קבצים

4. מבנה הפרויקט

Vaad_M_H/
├── frontend/              # React Frontend
│   ├── src/
│   │   ├── components/   # קומפוננטות
│   │   ├── pages/        # עמודים
│   │   ├── context/      # React Context
│   │   └── api/          # API client
│   └── public/           # קבצים סטטיים
├── backend/              # Express Backend
│   ├── src/
│   │   ├── routes/       # נתיבי API
│   │   ├── middleware/   # Middleware
│   │   └── scripts/      # סקריפטים
│   └── prisma/           # מסד נתונים
└── docker-compose.yml    # הגדרות Docker
