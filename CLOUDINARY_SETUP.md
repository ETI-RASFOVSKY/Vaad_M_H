# מדריך הגדרת Cloudinary - שלב אחר שלב

## מה זה Cloudinary?

Cloudinary הוא שירות אחסון תמונות וסרטונים בענן. האתר משתמש בו כדי לשמור את התמונות והסרטונים שאת מעלה.

## 🎯 שלב 1: הירשמות ל-Cloudinary

1. לך ל-[cloudinary.com](https://cloudinary.com)
2. לחץ על **"Sign Up"** (הירשם)
3. מלא את הפרטים:
   - שם פרטי
   - אימייל
   - סיסמה
4. אשר את האימייל שלך

## 🔑 שלב 2: מציאת הפרטים

לאחר ההתחברות:

1. **לך ל-Dashboard** - זה המסך הראשי
2. **תראה שם 3 דברים חשובים:**
   - **Cloud Name** - למשל: `dxy8k9abc` (זה השם של החשבון שלך)
   - **API Key** - מספר ארוך, למשל: `123456789012345`
   - **API Secret** - מפתח סודי, למשל: `abcdefghijklmnopqrstuvwxyz123456`

3. **איפה למצוא:**
   - בדרך כלל זה מופיע ישירות ב-Dashboard
   - אם לא, לחץ על **"Settings"** (הגדרות) → **"Security"** (אבטחה)

## 📝 שלב 3: העתקת הפרטים לקובץ

1. פתח את הקובץ: `C:\Users\PC\Desktop\vaad_2\Vaad_M_H\backend\.env`

2. מצא את השורות:
   ```
   CLOUDINARY_CLOUD_NAME=""
   CLOUDINARY_API_KEY=""
   CLOUDINARY_API_SECRET=""
   ```

3. מלא את הפרטים:
   ```
   CLOUDINARY_CLOUD_NAME="dxy8k9abc"
   CLOUDINARY_API_KEY="123456789012345"
   CLOUDINARY_API_SECRET="abcdefghijklmnopqrstuvwxyz123456"
   ```
   
   **⚠️ חשוב:** החלף את הערכים בדוגמה בערכים האמיתיים שלך!

4. שמור את הקובץ

## 🔄 שלב 4: הפעלה מחדש

1. **עצור את ה-backend** (לחץ Ctrl+C בטרמינל)
2. **הרץ שוב:**
   ```bash
   cd backend
   npm run dev
   ```

## ✅ בדיקה

1. לך ל-Admin Panel: `http://localhost:3000/admin/login`
2. התחבר
3. לך ל-"גלריה"
4. נסה להעלות תמונה
5. אם זה עובד - הכל מוכן! 🎉

---

## 💡 טיפים:

- **Cloudinary חינמי** עד 25GB אחסון
- **הפרטים שלך בטוחים** - רק האתר שלך משתמש בהם
- **אפשר להעלות תמונות גדולות** - Cloudinary מטפל בהכל

## ❓ בעיות?

**"שגיאה בהעלאת תמונה"**
- בדוק שהפרטים ב-`.env` נכונים
- ודא שהפעלת מחדש את ה-backend
- בדוק שיש חיבור לאינטרנט

**"לא רואה את הפרטים ב-Dashboard"**
- ודא שהתחברת לחשבון הנכון
- נסה להתנתק ולהתחבר שוב
