# הגדרת מייל מהירה - Resend (5 דקות)

## שלבים מהירים:

1. **היכנסו ל-Resend.com** ויצרו חשבון חינמי
2. **לכו ל-API Keys** ויצרו API Key חדש
3. **העתיקו את ה-API Key** (מתחיל ב-`re_`)
4. **הוסיפו ל-`backend/.env`:**

```env
RESEND_API_KEY=re_your_api_key_here
EMAIL_FROM=onboarding@resend.dev
ADMIN_EMAIL=e0548451274@gmail.com
```

5. **הפעילו מחדש את השרת** - זהו! ✅

---

## זה הכל!

עכשיו כל המיילים יעבדו:
- ✅ מיילים למנהל מהודעות יצירת קשר
- ✅ מיילי אישור למשתמשים
- ✅ קודי אימות במייל
- ✅ קודי איפוס סיסמה

**אין צורך ב-App Password או הגדרות מורכבות!**

---

## אם תרצו להשתמש בדומיין משלכם:

1. ב-Resend, לכו ל-"Domains"
2. הוסיפו את הדומיין שלכם
3. הוסיפו את ה-DNS records שהמערכת מבקשת
4. אחרי האימות, שינוי את `EMAIL_FROM` ל-`noreply@yourdomain.com`
