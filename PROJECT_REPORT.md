## מה נבנה

### Frontend (חלק המשתמש)

#### עמודים ציבוריים:

1. **עמוד הבית** (`/`)
   - וידאו רקע אוטומטי (ללא קול, בלולאה)
   - כותרת ושורת חזון
   - כפתורי CTA (קרא עוד, תרומה)
   - סטטיסטיקות אנימטיביות:
     - 15+ ועדים פעילים
     - 10+ ערים
     - 500+ משתתפים
     - 5+ שנות פעילות

2. **אודותינו** (`/about`)
   - מידע מפורט על הארגון
   - הסיפור והקמת הארגון
   - תחומי פעילות:
     - ועדים לבחורים
     - ועדים לאברכים
     - שבתות גיבוש
     - אירועים מיוחדים (חנוכה, ל"ג בעומר)
   - הצוות והרבנים

3. **גלריה** (`/gallery`)
   - תצוגת תמונות וסרטונים
   - סינון לפי קטגוריות:
     - הכל
     - תמונות
     - סרטונים
     - חגים (חנוכה, ל"ג בעומר, וכו')
   - Lightbox לצפייה בגדול
   - Infinite scroll

4. **צור קשר** (`/contact`)
   - טופס יצירת קשר עם ולידציה
   - שדות: שם, אימייל, תוכן
   - בדיקת תקינות
   - הודעת אישור לשליחה

5. **תרומה** (`/donate`)
   - טקסט מעודד תרומה
   - קישור ישיר לנדרים פלוס
   - הסבר על חשיבות התרומה

### מערכת ניהול (Admin Panel)

1. **התחברות מנהל** (`/admin/login`)
   - מסך login עם אימייל וסיסמה
   - JWT Authentication
   - אבטחה מלאה

2. **לוח בקרה** (`/admin`)
   - סטטיסטיקות:
     - מספר הודעות
     - הודעות לא נקראו
     - מספר תמונות/סרטונים
     - פירוט לפי סוג

3. **ניהול גלריה** (`/admin/media`)
   - העלאת תמונות וסרטונים
   - שיוך לקטגוריה
   - הוספת כותרת ותיאור
   - מחיקה והסרה
   - תצוגה גרפית

4. **ניהול הודעות** (`/admin/messages`)
   - צפייה בכל ההודעות
   - סינון: הכל, לא נקראו, נקראו
   - סימון כטופל
   - תצוגה לפי תאריך

## טכנולוגיות

### Frontend

- **React 18** - ספריית UI מודרנית
- **TypeScript** - תכנות בטוח וברור
- **Vite** - כלי בנייה מהיר
- **Tailwind CSS** - עיצוב מהיר ורגיש
- **Framer Motion** - אנימציות חלקות
- **React Router** - ניווט בין עמודים
- **React Hook Form + Zod** - טופסים עם ולידציה
- **Axios** - תקשורת עם שרת

### Backend

- **Node.js 20** - סביבת הרצה
- **Express.js** - שרת API
- **Prisma ORM** - ניהול מסד נתונים
- **PostgreSQL 15** - מסד נתונים יחסי
- **JWT** - אימות משתמשים
- **bcryptjs** - הצפנת סיסמאות
- **Cloudinary** - אחסון מדיה בענן
- **Multer** - העלאת קבצים
- **express-validator** - ולידציה

### DevOps

- **Docker** - מיכליזציה
- **Docker Compose** - ניהול מיכלים
- **Git** - בקרת גרסאות
- **Nginx** - שרת רשת (לייצור)

## מסד נתונים

### PostgreSQL

**טכנולוגיה:** PostgreSQL 15  
**ORM:** Prisma  
**מיקום:** Docker Container או Render

### טבלאות

#### 1. Users (משתמשי מנהל)

| שדה | סוג | תיאור |
|-----|-----|-------|
| id | UUID | מזהה ייחודי |
| email | String | אימייל (ייחודי) |
| passwordHash | String | סיסמה מוצפנת |
| role | String | תפקיד (admin) |
| createdAt | DateTime | תאריך יצירה |
| updatedAt | DateTime | תאריך עדכון |

#### 2. Messages (הודעות)

| שדה | סוג | תיאור |
|-----|-----|-------|
| id | UUID | מזהה ייחודי |
| name | String | שם השולח |
| email | String | אימייל השולח |
| content | Text | תוכן ההודעה |
| handled | Boolean | האם טופל |
| createdAt | DateTime | תאריך יצירה |

#### 3. Media (תמונות/סרטונים)

| שדה | סוג | תיאור |
|-----|-----|-------|
| id | UUID | מזהה ייחודי |
| url | String | כתובת המדיה |
| cloudinaryId | String | מזהה ב-Cloudinary |
| type | String | סוג (image/video) |
| category | String | קטגוריה |
| title | String | כותרת |
| description | Text | תיאור |
| createdAt | DateTime | תאריך יצירה |
