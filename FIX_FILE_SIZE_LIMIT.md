# תיקון גבול גודל קובץ

## הבעיה

כשמנסים להעלות קובץ גדול מדי, מקבלים שגיאה:
```
MulterError: File too large
```

## מה תוקן

1. **הגדלתי את הגבול מ-50MB ל-200MB** - עכשיו אפשר להעלות קבצים עד 200MB
2. **שיפרתי את הודעת השגיאה** - עכשיו יש הודעה בעברית שמסבירה את הבעיה

## איך זה עובד עכשיו

- **גודל מקסימלי:** 200MB
- **סוגי קבצים מותרים:** תמונות (jpeg, jpg, png, gif) וסרטונים (mp4, mov, avi, webm)
- **הודעת שגיאה:** אם הקובץ גדול מדי, תקבלו הודעה בעברית: "הקובץ גדול מדי. הגודל המקסימלי הוא 200MB. אנא נסו קובץ קטן יותר או דחוסו את הקובץ."

## אם צריך להגדיל עוד יותר

אם תרצו להגדיל את הגבול עוד יותר, ערכו את `backend/src/routes/media.js`:

```javascript
limits: {
  fileSize: 200 * 1024 * 1024, // שנה את 200 ל-500 או יותר (ב-MB)
},
```

**שימו לב:** 
- Cloudinary גם יש לו גבולות - בדקו את התוכנית שלכם
- קבצים גדולים יותר לוקחים יותר זמן להעלות
- אם אתם משתמשים ב-memory storage, קבצים גדולים יכולים להשתמש בהרבה זיכרון

## אם עדיין יש בעיה

אם הקובץ שלכם קטן מ-200MB ועדיין מקבלים שגיאה:
1. בדקו את הגודל האמיתי של הקובץ
2. נסו לדחוס את הקובץ
3. בדקו את הגבולות של Cloudinary
