## 🎥 איך להוסיף סרטון רקע?

### שלב 1: קח סרטון
- פורמט: `.mp4`
- גודל: עד 50MB (ככל שקטן יותר, כך טוב יותר)
- אורך: 10-30 שניות
- **ללא קול** (הסרטון רץ בלי קול)

### שלב 2: העתק את הקובץ
1. קח את קובץ הסרטון (למשל: `my-video.mp4`)
2. העתק אותו לתיקייה:
   ```
   C:\Users\PC\Desktop\vaad_2\Vaad_M_H\frontend\public\
   ```
3. **שנה את השם ל:** `hero-video.mp4`

**זה הכל!** הסרטון יופיע אוטומטית ברקע עמוד הבית.

## 🖼️ איך להעלות תמונות?

### ⚠️ חשוב להבין:
**נדרים פלוס זה רק לתרומות!**  
תמונות מעלים דרך **מערכת הניהול של האתר שלך**, לא דרך נדרים פלוס.

### איך לעשות:

1. **הרץ את האפליקציה:**
   ```bash
   npm run dev
   ```

2. **פתח דפדפן ולך ל:**
   ```
   http://localhost:3000/admin/login
   ```

3. **התחבר:**
   - אימייל: `admin@vaad.org`
   - סיסמה: `admin123`

4. **לך ל-"גלריה"** בתפריט העליון

5. **לחץ על "העלה מדיה חדשה"**

6. **בחר תמונה** מהמחשב

7. **התמונה תתחיל להעלות אוטומטית!**

8. **התמונה תופיע** בגלריה הציבורית: `http://localhost:3000/gallery`

### ⚠️ אבל קודם - הגדר Cloudinary:

כדי שהעלאת תמונות תעבוד, צריך חשבון Cloudinary (חינמי):

1. **הירשם ל:** [cloudinary.com](https://cloudinary.com)
2. **לאחר ההתחברות**, תראה ב-Dashboard:
   - Cloud Name
   - API Key  
   - API Secret
3. **פתח את הקובץ:** `C:\Users\PC\Desktop\vaad_2\Vaad_M_H\backend\.env`
4. **מצא את השורות:**
   ```
   CLOUDINARY_CLOUD_NAME=""
   CLOUDINARY_API_KEY=""
   CLOUDINARY_API_SECRET=""
   ```
5. **מלא את הפרטים** (בין המרכאות)
6. **שמור את הקובץ**
7. **הפעל מחדש את ה-backend** (Ctrl+C ואז `npm run dev`)
