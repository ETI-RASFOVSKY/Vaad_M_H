# 🎥 מדריך הוספת סרטון רקע - שלב אחר שלב

## איך להוסיף סרטון שמסתובב בלולאה?

האתר כבר מוכן להציג סרטון רקע! כל מה שצריך זה להוסיף את קובץ הסרטון.

### שלב 1: הכנת הסרטון

**דרישות:**
- ✅ פורמט: `.mp4` (מומלץ)
- ✅ גודל: עד 50MB (כדי שהאתר יטען מהר)
- ✅ אורך: 10-30 שניות (מומלץ)
- ✅ ללא קול (הסרטון רץ בלי קול)

**טיפים:**
- השתמש ב-HandBrake או כל תוכנה אחרת לדחיסת הסרטון
- רזולוציה מומלצת: 1920x1080 (Full HD)
- נסה לשמור על גודל קטן ככל האפשר

### שלב 2: העתקת הקובץ

1. **קח את קובץ הסרטון** (למשל: `hero-video.mp4`)
2. **העתק אותו** לתיקייה:
   ```
   C:\Users\PC\Desktop\vaad_2\Vaad_M_H\frontend\public\
   ```
3. **שם את הקובץ:** `hero-video.mp4`

**חשוב:** השם חייב להיות בדיוק `hero-video.mp4`!

### שלב 3: בדיקה

1. **הרץ את האפליקציה:**
   ```bash
   npm run dev
   ```

2. **פתח את האתר:** `http://localhost:3000`

3. **הסרטון אמור להופיע** ברקע עמוד הבית!

### איך זה עובד?

- ✅ הסרטון **מתחיל אוטומטית** כשנכנסים לעמוד
- ✅ הסרטון **ללא קול** (muted)
- ✅ הסרטון **מסתובב בלולאה** (loop)
- ✅ אם הסרטון לא נטען, יופיע רקע אפור יפה

## 🖼️ איך להעלות תמונות לגלריה?

### דרך Admin Panel (הדרך הנכונה):

**נדרים פלוס זה רק לתרומות!** תמונות מעלים דרך מערכת הניהול של האתר שלך.

### שלב 1: התחברות

1. לך ל: `http://localhost:3000/admin/login`
2. התחבר עם: `admin@vaad.org` / `admin123`

### שלב 2: העלאת תמונה

1. **לחץ על "גלריה"** בתפריט העליון
2. **לחץ על "העלה מדיה חדשה"**
3. **בחר קובץ** - לחץ ובחר תמונה מהמחשב
4. (אופציונלי) בחר **קטגוריה**: חנוכה, ל"ג בעומר, שבת גיבוש, כללי
5. (אופציונלי) הוסף **כותרת** ו**תיאור**
6. **הקובץ יתחיל להעלות אוטומטית!**

### שלב 3: התמונה מופיעה

התמונה תופיע:
- ✅ בגלריה הציבורית: `http://localhost:3000/gallery`
- ✅ עם אפשרות לראות אותה בגדול (Lightbox)
- ✅ עם סינון לפי קטגוריות

## ⚠️ חשוב - Cloudinary

כדי להעלות תמונות, צריך חשבון Cloudinary.

**איך להגדיר:**
1. הירשם ל-[cloudinary.com](https://cloudinary.com) (חינמי)
2. קבל את הפרטים: Cloud Name, API Key, API Secret
3. הוסף ל-`backend/.env`:
   ```
   CLOUDINARY_CLOUD_NAME="your-cloud-name"
   CLOUDINARY_API_KEY="your-api-key"
   CLOUDINARY_API_SECRET="your-api-secret"
   ```
4. הפעל מחדש את ה-backend

**ללא Cloudinary:**
- ❌ לא תוכל להעלות תמונות
- ✅ אבל תוכל לראות את כל שאר האתר
